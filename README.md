# wkhtmltopdf cookbook

Installs wkhtmltopdf from binary.

# Requirements

`apt` and Ubuntu.

# Usage

Include cookbook in your `Berksfile`.

````
cookbook 'wkhtmltopdf', github: 'aerogami-cookbooks/wkhtmltopdf'
````

Then install.

````
berks install
````
